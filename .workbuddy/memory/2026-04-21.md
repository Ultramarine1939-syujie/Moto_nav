# 2026-04-21 日志

## moto-nav_v2 项目 - 途径点功能

### 已完成
- 添加"添加途径点"按钮，点击后在起点终点之间插入途径点输入框
- 支持最多3个途径点
- 途径点以橙色数字标记显示在地图上
- 路线规划改为**分段规划**模式：
  - 起点→途径点1、途径点1→途径点2、...、最后途径点→终点
  - 每段独立调用高德API规划
  - 合并显示总距离、总时间和各段指引

### 修改的文件
- `index.html` - 添加途径点容器和按钮
- `css/style.css` - 添加途径点样式、分段标题样式
- `js/config.js` - 添加waypoint颜色配置
- `js/marker.js` - 添加setWaypoint、clearWaypoint等方法
- `js/map.js` - 添加pendingWaypointIndex状态处理
- `js/ui.js` - 添加途径点UI逻辑、displayRoute支持多段
- `js/route.js` - 重构为分段规划（planMultiSegment、drawMultiSegment）
